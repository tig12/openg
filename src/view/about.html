About the Open Gauquelin Database

The purpose of the Open Gauquelin database is to support statistical works done to test astrology.
Written by Thierry Graff - this is a personal project, done on my spare time.
You can contact me by email :

This site doesn't use cookies (GPDR compliant).

The database is generated by a program called gauquelin5 (aka g5).
  • Code repository - github.com/tig12/gauquelin5
  • Documentation - tig12.github.io/gauquelin5
Current site, to browse the database, is managed by a program called openg.
  • Code repository - github.com/tig12/openg
