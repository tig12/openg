How to add or correct open Gauquelin data ?
Abbreviations
  • BC: Birth Certificate
  • ogdb: Open Gauquelin Data Base
To participate to the construction of a reliable database, verified by humans:
  1. Choose a person from the database or a new person.
  2. Visit the online civil registry containing the birth certificate of the person.
  3. Fill a document, DC.yml (see below), containing:
    • A transcription of the informations contained in the BC.
    • Precise indications permiting to easily consult the BC online.
  4. Optionally, retrieve image(s) of the original act.
  5. Send it for addition in ogdb.
    • Send the files by email (see Contact)
    • If you have a github account, you can make a pull request in the ogdb wiki repository
