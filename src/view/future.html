Future developments

Notation: g5: Program used to build the ogdb.
From tig12.github.io/gauquelin5:
  1. Gather the historical data, check them, correct them, and merge them in a single database.
  2. Connect the historical data to standard datasets like Wikidata and complete birth times to build larger groups.
  3. Build new eminence indicators and sort famous persons by eminence.
Current state of the code only deals with step 1.
The purpose is to have a 1.0 version with a complete restoration of mars effect data, to permit a reproduction of Suitbert Ertel's analysis contained in the book "The Tenacious Mars Effect".
